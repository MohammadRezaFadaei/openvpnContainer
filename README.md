# OpenVPN and Docker
Implement a containerized OpenVPN with an interactive-CLI interface.

## How to use
Clone the repository and use:
```bash
git clone https://github.com/MohammadRezaFadaei/openvpnContainer.git
cd openvpnContainer
sudo ./install.sh
```

## Advantages
* Easy to use
* A to Z automated
* Haveing a comfortable a interface
